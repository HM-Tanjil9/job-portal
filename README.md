**Intelligent Job Matching Platform | Scalable Microservices Architecture**
###### A fully scalable, real-world Job Portal built with a Microservices Architecture. This platform integrates AI-driven career tools, real-time messaging, and a robust subscription model to bridge the gap between recruiters and candidates.
</div>

---

## 🎯 Key Features


🤖 AI Career Guide: Get suggetions based on your skills, get score of your CV

👥 Multi-Role System: Dedicated flows for Jobseeker and Recruiters.

📈 Recruiter Dashboard: Comprehensive tools to manage job postings, track applicants, and view analytics.

💳 Payment: Integrated sslcommerz sandbox subscription system for premium features.

🔔 Real-time Notifications: Automated email triggers via Kafka event workers.

🔒 Secure Auth: Full authentication suite including Forgot/Reset password flows and profile management (Bio, Skills).

##  📐 Microservices Overview

Auth Service: Handles JWT-based authentication and user management.

Job Service: Manages job listings, search functionality, and applications.

AI Service: Dedicated worker for Resume Analysis and Career Guidance.

Notification Service: Kafka consumer for sending real-time emails.

Payment Service: Manages sslcommerz sandbox and subscription states.
